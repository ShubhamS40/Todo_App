# Todo-App 

A Simple Basic Todo Webpage   Built with  React.js
and Tailwindcss that is basically a Todo App  its Note Your Daily Task and Target.

## Features

- Note Your Daily Task
- After Done You Can Checkout
- Responsive 
- Edit Your Task 
- Delete Completed Task
